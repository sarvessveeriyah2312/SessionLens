# Contributing

Contributions are welcome!

## How to Contribute

1. Fork the repository
2. Create a feature branch
   ```bash
   git checkout -b feature/amazing-feature
   ```
3. Commit your changes using [Conventional Commits](https://www.conventionalcommits.org/)
   ```bash
   git commit -m "feat: add amazing feature"
   ```
4. Push to your branch
   ```bash
   git push origin feature/amazing-feature
   ```
5. Open a Pull Request

## Guidelines

- Follow the existing code style
- Use Conventional Commits for all commit messages (this drives automated versioning)
- Test your changes with `npm run dev` before submitting

## Reporting Issues

- [Report a bug](https://github.com/sarvessveeriyah2312/SessionLens/issues)
- [Request a feature](https://github.com/sarvessveeriyah2312/SessionLens/issues)
